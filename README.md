Generador de paletas de colores 

-Descripción
Este proyecto consiste en una página web que genera 6, 8 o 9 cuadrados de colores utilizando el modelo HSL y HEX

-Objetivo
Practicar HTML, JavaScript, manipulación del DOM, utilización de GitHub y estilos con CSS.

-Tecnologías utilizadas
HTML
CSS
JavaScript

-Pasos que seguí
Creé la estructura básica en HTML.
Agregué un contenedor para los cuadrados.
Usé CSS Grid para organizar los elementos en una grilla de 3x2, 4x2 y 3x3.
Implementé JavaScript para generar dinámicamente los cuadrados.
Agregué 2 botones para poder cambiar el número de paletas y el formato de colores.
Utilicé la función hsl y hex de manera random para asignar diferentes colores.
Generé un toast al hacer click en el botón principal de "Generar paleta"
Agregué la función del copiado al portapapeles al hacer click sobre un color
Modifiqué el código para poder verlo desde un dispositivo celular o tablet

-Cómo usar el proyecto
Abrir el archivo index.html en el navegador.
Elegir cuántas paletas de colores queremos ver (6, 8 o 9 colores)
Seleccionar el formato de colores deseado (HSL o HEX)
Hacer click en el botón "generar paleta"
Visualizar los cuadrados generados automáticamente.
En caso de querer copiar el color al portapapeles, hacer click sobre cualquier paleta.
